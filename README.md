# Iron Core Nutrition

A premium, dark-only performance nutrition landing page built with Next.js, React, Tailwind CSS, and Framer Motion.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Production build

```bash
pnpm build
pnpm start
```

## Included

- Responsive desktop, tablet, and mobile layouts
- Sticky glass navbar and mobile menu
- Scroll progress, reveal animations, parallax, floating visuals, and hover/tap motion
- Hero, products, benefits, flagship spotlight, stacks, training, athletes, reviews, CTA, and footer
- Reduced-motion accessibility support
- Original AI-generated product and athlete campaign imagery

## Generated image prompts

The built-in image generation tool was used for two project assets:

1. A photorealistic premium lineup of five matte-black supplement containers in a black studio, lit with red and blue rim lights, with no real logos or readable branding.
2. A photorealistic wide sports campaign triptych featuring a bodybuilder, strength coach, and fitness athlete in a dark studio with red/blue edge lighting and no logos or text.
