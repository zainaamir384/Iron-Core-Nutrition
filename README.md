# Iron Core Nutrition

Iron Core Nutrition is a modern frontend landing page project based on a gym supplements and performance nutrition brand concept.

**Frontend Project Designed & Developed by Zain Aamir**

This project was created to practice modern UI design, responsive layouts, dark theme styling, smooth animations, product sections, and clean frontend development.

## Project Overview

The website uses a premium black theme with red and blue accents. It includes a modern landing page layout for supplement products, benefits, product stacks, athlete-style sections, reviews, and a final call-to-action area.

This is a frontend practice and portfolio project. It does not include backend, payment system, database, or real checkout functionality.

## Run Locally

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Open:

```text
http://localhost:3000
```

## Build

Create production build:

```bash
pnpm build
```

Start production server:

```bash
pnpm start
```
